# ☕ AGOYO Ninja

Game ketangkasan bertema kopi, terinspirasi dari Fruit Ninja. Iris biji kopi, cangkir espresso, gelas latte, dan kemasan kopi yang terlempar ke udara sebanyak mungkin sebelum nyawamu habis!

## Cara Bermain

1. Buka file `index.html` di browser (atau aktifkan GitHub Pages untuk main online).
2. Klik tombol **Mulai Main**.
3. Gunakan mouse (drag/geser) atau jari (di perangkat sentuh) untuk mengiris objek kopi yang melayang.
4. Setiap objek kopi yang berhasil diiris menambah skor.
5. Jangan mengiris **Bom Kopi Panas** 💣, atau permainan langsung berakhir.
6. Objek kopi yang jatuh tanpa diiris akan mengurangi nyawa. Kehabisan nyawa = Game Over.
7. Kecepatan dan jumlah objek akan meningkat seiring waktu bermain.

## Fitur

- Grafis sederhana berbasis HTML5 Canvas, tanpa dependency eksternal.
- Efek jejak irisan (slice trail) mengikuti gerakan mouse/jari.
- Sistem skor, nyawa, dan skor terbaik (tersimpan di localStorage).
- Tingkat kesulitan meningkat secara dinamis.
- Mendukung mouse dan layar sentuh (mobile friendly).

## Menjalankan Secara Lokal

Cukup buka file `index.html` langsung di browser, tidak perlu server atau instalasi apa pun.
